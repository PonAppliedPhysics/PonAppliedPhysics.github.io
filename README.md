# Pon Aung Portfolio Site

This folder contains a single-page GitHub Pages portfolio.

## Files
- `index.html` — main portfolio website with clickable project detail modals
- `Engineering_Portfolio.pdf` — supporting portfolio PDF linked from the site

## How to use on GitHub Pages
1. Create a GitHub repository, for example `portfolio` or `PonAppliedPhysics.github.io`.
2. Upload `index.html` and `Engineering_Portfolio.pdf` to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Wait a minute or two for GitHub Pages to publish the site.

Clicking a project card opens a detailed project modal. The project modals also support direct links using hashes such as `#project-honda`, `#project-weather`, and `#project-thermal`.
